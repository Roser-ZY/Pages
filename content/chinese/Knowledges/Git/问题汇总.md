### rpc error

![](image/rpc%20error.png)

这种情况一般是网络问题或者文件过大导致下载失败。可以通过设置浅克隆来解决。

```shell
git fetch --depth=50
```

如果是推送时出现该错误，则可能是数据量过大导致，可以扩大缓冲区。

```shell
git config --global http.postBuffer 524288000
```

上述命令设置缓冲区为 500MB。

- $ 如果还是有问题，建议改为 SSH 的方式，这是最简单的方法，不会存在缓冲区问题。

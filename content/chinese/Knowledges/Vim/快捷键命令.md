### 翻页

| 操作       | 功能      | 备注       |
| -------- | ------- | -------- |
| `Ctrl-d` | 向下翻半页。  | Down     |
| `Ctrl-f` | 向下翻一整页。 | Forward  |
| `Ctrl-u` | 向上翻半页。  | Up       |
| `Ctrl-b` | 向上翻一整页。 | Backword |
### 切换标签页

| 操作      | 功能           |
| ------- | ------------ |
| `gt`    | 切换到下一个标签页。   |
| `gT`    | 切换到上一个标签页。   |
| `{n}gt` | 切换到第 n 个标签页。 |
### 前进后退

| 操作       | 功能  |
| -------- | --- |
| `Ctrl-o` | 后退  |
| `Ctrl-i` | 前进  |
